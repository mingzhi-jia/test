# Linux  日期指令

## date

> 查看及修改日期时间

输出年月日时分秒：  `# date +"%Y-%m-%d %H:%M:%S"`   或  `date +"%F %T"`

| 选项 | 全称 | 作用                                                         |
| ---- | ---- | ------------------------------------------------------------ |
| -d   | date | 显示字符串所指定的日期和时间<Br>eg：`# date -d "-1 day" +"%F %T"`  <br>eg:   `# date -d "+1 year" +"%F %T"` |
| -s   | set  | 设置日期<br>eg:`# date -s '2028-10-01'` 、 `# date -s '10:10:00'` |

#### 常用日期格式字符串:

| 字符 | 作用                    |
| ---- | ----------------------- |
| %Y   | 年                      |
| %m   | 月                      |
| %d   | 日                      |
| %H   | 小时，24小时制（00~23） |
| %I   | 小时，12小时制（01-12） |
| %M   | 分，（0~59）            |
| %S   | 秒，（00~59）           |
| %F   | 年月日（%Y-%m-%d）      |
| %T   | 时分秒（%H:%M:%S）      |

## hwclock

> 查看或设置硬件时间（主板上的时间）

| 选项 | 全称    | 作用                   |
| ---- | ------- | ---------------------- |
| -s   | hctosys | 设置系统时间为硬件时间 |
| -w   | systohc | 设置硬件时间为系统时间 |
